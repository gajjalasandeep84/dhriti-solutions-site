=== Consulting Services React App ===

1. LOCAL SETUP:
---------------
- Install Node.js (https://nodejs.org)
- Run:
  > npm install
  > npm start
- Your app will be available at http://localhost:3000

App Structure:
- Header, About, Services, Contact, Footer

2. DEPLOYMENT:
---------------
Push to GitHub and deploy using Vercel or Netlify with default React build commands.
